Blotato API — What's Now Possible

Full capability map from the live API docs. Current integration uses ~5% of this.

11 Platforms via one API key
Twitter / X Instagram TikTok YouTube LinkedIn Facebook Threads Bluesky Pinterest Webhook
5 Unlocked Capabilities
📤

1. Multi-Platform Publish

POST /v2/posts — one slot → 8+ platforms. Per-platform captions already live in DB (caption_ig, caption_tiktok, caption_yt, caption_x, caption_linkedin). Wire them up.

✓ Currently: text-only, no platform targeting
→ Now: per-platform captions + account ID
🎬

2. Video Autopublish

POST /v2/media → upload filmed reel to Blotato CDN → attach mediaUrl to post. TikTok, IG Reels, YT Shorts all accept video. Film once, auto-post everywhere.

✓ Currently: captions only
→ Now: actual video posted automatically
🖼️

3. AI Visual Generation

POST /v2/videos/from-templates — script → Blotato auto-generates Instagram Carousel, Quote Cards, Tweet Cards, Tutorial Slides, AI Story Video. Replaces Canva.

✓ Currently: manual design
→ Now: carousel from script in 1 click
👤

4. Account Sync

GET /v2/users/me/accounts — pull all connected accounts (with IDs) into DB. UI shows checkboxes: pick which accounts to publish each slot to. Subaccounts for FB pages / LinkedIn pages.

✓ Currently: hardcoded, no account selection
→ Now: per-slot account picker
⏱️

5. Smart Scheduling

scheduledTime (ISO 8601) OR useNextFreeSlot: true. Blotato auto-picks the next open slot in your posting schedule. Content Hub calendar drives the schedule_date, Blotato fires it.

✓ Currently: manual mark-as-posted only
→ Now: fully automated fire-and-forget
What Blotato does NOT cover

No analytics pull API (use YouTube Data API + platform-native for stats). No inbox/comments management. No auto-transcription (keep Whisper). Video generation is template-based, not AI from raw footage.
