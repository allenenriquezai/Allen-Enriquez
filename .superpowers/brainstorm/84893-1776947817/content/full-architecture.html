3 Automated Flows (n8n triggers Content Hub)
FLOW 1 — AUTO PUBLISH n8n cron checks every 15min for due slots
Slot due in calendar → n8n hits /api/blotato/publish → Upload video to Blotato CDN → 1 post per platform (IG, TikTok, YT, FB, LinkedIn, Threads) → Slot marked "posted"
Per-platform captions already in DB (caption_ig, caption_tiktok, caption_yt, caption_x, caption_linkedin). Each post uses its own caption + platform-specific settings (TikTok privacy, YT title/privacy, IG Reels settings).
FLOW 2 — AUTO ANALYTICS n8n cron daily at 6am
n8n triggers /api/analytics/refresh → YouTube Analytics API (already done) + Meta Insights API (IG + FB — 1 token) + TikTok basic metrics → All into SQLite metrics table
Replaces manual metric entry. Analytics tab shows real data automatically. Cross-platform reach + engagement in one view.
FLOW 3 — AUTO INBOX n8n cron every 30min
n8n triggers /api/inbox/sync → IG comments (Meta API) + FB comments (Meta API) + YouTube comments → inbox table + AI reply suggestions
Claude drafts a reply suggestion per comment. One-click send via Meta Graph API / YouTube API. TikTok comments: read-only for now. X: skip (no free API).
