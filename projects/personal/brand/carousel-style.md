# Carousel Style Reference

## Primary Reference: Dan Martell (@danmartell)

Style: clean, minimal, massive text on white background.

### Key traits:
- White background, no gradients, no images
- Very large bold text — fills 60-70% of the slide width
- Clean sans-serif font (not condensed/narrow like Impact)
- Black text for main points, dark gray for secondary
- Minimal elements — text only, maybe a small accent
- Left-aligned, not centered horizontally
- Vertically centered in the middle of the slide
- Max 15-20 words per slide
- One idea per slide

### Allen's additions:
- Brand blue (#02B3E9) accent bar at top of title/CTA slides
- Brand blue accent bar between header and body on content slides
- Brand blue swipe dots
- @allenenriquez handle on title and CTA slides
- Slide counter (e.g. 1/4) on content slides

### Tool: `tools/personal/generate_carousel.py`
- Default style matches this reference
- 1080x1350px (4:5 ratio for IG/FB)
- Helvetica Bold for all text
- Auto-scales font down if text overflows
- Copy uses `||` separator for header/body split
