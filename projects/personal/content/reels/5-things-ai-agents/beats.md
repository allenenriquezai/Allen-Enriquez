## Animation upgrades vs reel-2

1. **Big numerals** — `01`–`05` slam in on each item with chromatic-split glitch (better than reel-2's static banners).
2. **Per-scene UI variety** — inbox / calendar / dashboard / invoices / doc expansion. Each distinct, data-feel.
3. **Transitions rotate** — pop+flash between retention→#1, whip-pan between #3→#4, flash-through-white into payoff.
4. **Secondary motion** — all scenes have continuous drift (cursor blink, glow pulse, ticker scroll, shimmer) through full duration.
5. **Payoff scene = jaw-dropper** — dual clock animation + chromatic slam on "60×".
6. **Audio-reactive headings** — 3–6% pulse on scene numeral on each "Number X" word.
