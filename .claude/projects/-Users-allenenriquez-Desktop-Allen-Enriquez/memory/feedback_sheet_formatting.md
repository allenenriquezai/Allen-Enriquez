Don't change column widths/sizes in Google Sheets. Allen manually adjusts them for readability (bad eyesight, zooms in).

**Why:** After the CRM cleanup auto-resized columns, Allen had to manually fix the sizes.

**How to apply:** When formatting sheets, skip autoResizeDimensions. Only change colours, fonts, data validation, conditional formatting — not dimensions.
