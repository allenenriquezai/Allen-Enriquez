## Principles (seed)

- Phase 1: done-for-you AI sales system setup. First 3-5 clients free for testimonials.
- Pricing benchmark: $2K-$8K setup + $500-$1K/mo retainer.
- Soft CTA: "Everything is free. If you don't have time, I'll set it up for you."
- Niche: service businesses (trades, real estate, coaching). PH market first → AU/US/UK at Western rates.
- Delivery: scope discovery → build → QA gate → handoff. Never ship without QA.
- No paid API call without scope confirmation.
